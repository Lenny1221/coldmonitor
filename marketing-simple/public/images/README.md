# Foto's & screenshots voor de site

Plaats de bestanden hieronder in deze map (`marketing-simple/public/images/`) met
**exact** de vermelde bestandsnaam. Zolang een bestand ontbreekt, toont de site
automatisch een nette placeholder met de omschrijving — de layout blijft dus heel.

Tip: exporteer screenshots op een breed venster (geen privégegevens van klanten in beeld).

### Aanbevolen formaat
- Screenshots: PNG, 1600–2400 px breed.
- Foto's: JPG, 1600 px breed, < 500 KB (comprimeren mag, bv. via squoosh.app).
- Vermijd herkenbare klantnamen, prijzen van derden of persoonsgegevens in screenshots.
