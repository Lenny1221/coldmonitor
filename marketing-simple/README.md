# IntelliFrost — simpele marketing-site (lokaal)

Prototype: één pagina,zelfde kleuren en prijzen als de hoofdsite, veel compacter.

**Staat in `.gitignore`** — wordt niet naar GitHub gepusht.

## Starten

```bash
cd marketing-simple
npm install
npm run dev
```

Open http://localhost:5180

## Opmerking

Dit is los van `frontend/`. De bestaande app blijft ongewijzigd.
